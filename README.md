# 入力（タイピング）で戦う育成RPG

このリポジトリは、**タイピングが主軸**のミニRPGをPythonで実装したものです。  
入力の速さ・正確さがそのままダメージに変換され、短文/長文で戦い方が分かれます。

## 1. 設計

### ゲームループ
1. 問題文を出す（難易度はレベル連動）
2. プレイヤーが入力
3. 入力判定（正確さ・速度・長さ）
4. コンボ計算
5. ダメージ計算
6. 敵行動
7. 勝敗判定
8. 勝利時に成長処理（EXP→レベルアップ）

### コアシステム
- **入力判定 (`TypingJudge`)**
  - 一致判定
  - accuracy
  - cps（chars per second）
  - 短文/長文フラグ
- **コンボ計算 (`ComboSystem`)**
  - 正解短文でコンボ加算
  - ミスでコンボリセット＋弱体スタック
- **ダメージ計算 (`DamageCalculator`)**
  - `ATK × speed × accuracy × style × combo × weaken` を基礎
  - 長文は重撃ボーナス
  - 敵防御で軽減
- **成長処理 (`Character.gain_exp`)**
  - EXP累積
  - レベルアップでステータス上昇

## 2. 構成

- `game.py`
  - データ（問題文配列）
  - ステータス/キャラ定義
  - 入力判定・コンボ・ダメージ・戦闘・成長処理
  - 実行エントリ

## 3. コード（実行方法）

```bash
python3 game.py
```

起動後にスタイルを選択：
- `short`（短文型・高速）
- `long`（長文型・高火力）
- `balance`（バランス型）

## 4. 拡張方法

### 敵を増やす
- `BattleSystem` に敵テンプレート配列を追加し、ランダム/ステージ順で出現。

### 問題文を拡張
- `QUESTIONS` を難易度ごとに追加。
- テーマ（英単語/日本語ローマ字/コード断片）別プールも可能。

### 成長を深くする
- 特性（Trait）を複数化してパッシブ効果を追加。
- スキルツリー（短文特化・長文特化）を導入。

### プレイ感を改善
- 制限時間・クリティカル・状態異常を追加。
- ログ表示を色付け（`rich` など）して打鍵フィードバックを強化。
